## Install and Run Locally
```js
$ git clone https://github.com/sobitp59/remotework.git
$ cd remotework
$ pnpm install
$ pnpm run dev
```

## Features
- Search Jobs
- Bookmark Jobs
- Filter Jobs by Relevant and Recent
- Pagination

## Tech Stack Used
- NextJs
- TypeScript
- TailwindCSS
- React Query
- React Hot Toast
- Radix UI / Radix Icons
